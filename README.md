## Industry Benchmarking & Macroeconomic Analysis

**Tools:** Microsoft Excel  
**Sector:** Retail & Manufacturing

### What this project covers
- 15+ financial ratios across Liquidity, Leverage & Profitability
- Peer benchmarking: Monte Carlo, Trent, Titan
- Yield curve analysis: India, Pakistan, Japan

### Key Findings
- Monte Carlo leads on profitability (47% GM, 25.9% ROE)
- Trent leads on operational efficiency (4.95x Inventory Turnover)
- Pakistan yield curve signals sovereign distress
